Project Overview

The **Healthy Diet Affordability Dashboard** is an interactive R Shiny application designed to visualize and analyze data related to the cost and accessibility of a healthy diet in Pakistan. Leveraging comprehensive data, this tool empowers policymakers, researchers, and the public to explore crucial trends, identify key indicators, and understand the economic barriers to nutrition. It provides a data-driven foundation for informed decisions aimed at improving public health and food security.

Key Features

Intuitive Overview

Get instant insights into total indicators tracked, time periods covered, data points, and average values for key metrics.

Dynamic Trend Analysis

Explore how different healthy diet indicators change over time with interactive line, area, and bar charts.

Comprehensive Data Explorer

Browse, filter, and search the raw dataset efficiently, with options to download the data.

Bilingual Interface

Seamlessly switch between English and Urdu to cater to a broader audience.

Professional UI/UX

A clean, modern design with a calming color palette ensures an engaging and user-friendly experience.

Data-Driven Insights

Quick insights sections provide contextual summaries and highlight crucial trends from the data.

Technologies Utilized

  • R Shiny: Core framework for the interactive web application.
  • shinydashboard: For creating the structured dashboard layout.
  • dplyr & tidyr: Powerful tools for data manipulation and cleaning.
  • highcharter: For generating visually appealing and interactive charts.
  • DT: For robust and feature-rich interactive data tables.
  • rsconnect: For seamless deployment to shinyapps.io.
  • HTML, CSS (Custom & bslib): For custom styling and responsive design.

Deployment

This dashboard is deployed on shinyapps.io, ensuring public accessibility and reliable performance. The deployment process involved configuring the `rsconnect` package to seamlessly push the R Shiny application and its data to the cloud.
